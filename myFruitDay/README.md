# 部落格網站開發

### 使用技術
  * Python-Django 框架
  * MySQL
  * ORM 技術 
  * Cookie and Session 技術
  * Javascript-Ajax 跨域請求
  * Json 格式數據傳輸

### 功能說明

* 首頁
  * 商品列表
  	* 展示不同分類的商品, 購物車圖示點擊可加入購物車
  * 購物車頁面
  	* 展示商品訂單, 購物車商品的增、刪、改、查(CRUD)功能；修改商品數量或刪除時，商品價格部分刷新功能。 
  	修改數量時, 動態刷新每筆訂單金額與總金額。
  
  * 註冊
  	* 使用Ajax部分刷新 >>> 判斷帳號是否有人使用過
  * 登入
    * 判斷用戶是否註冊
